## Solution

- https://aerokube.com/

## Installations

> - Docker [https://docs.docker.com/]
- Text Editor
- Homebrew for mac [https://docs.brew.sh/Installation]

- setup k8s repo

```sh
$ helm repo add aerokube https://charts.aerokube.com/
$ helm repo update
$ helm upgrade --install --create-namespace -n moon moon aerokube/moon2
```
